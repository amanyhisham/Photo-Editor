
# 🖼️ Photo Editor

A browser-based **Photo Editor** that lets you upload and edit images using real-time CSS filters, built with **HTML**, **CSS**, and **JavaScript**.

---

## 🚀 Live Demo

🔗 [View Live](https://amanyhisham.github.io/Photo-Editor/)

---

## ✨ Features

- 📤 Upload any image from your device
- 🎨 Real-time image editing with 7 filter controls:
  - **Saturate** — adjust color intensity
  - **Contrast** — increase or decrease contrast
  - **Brightness** — make the image lighter or darker
  - **Sepia** — apply a warm vintage tone
  - **Grayscale** — convert to black & white
  - **Blur** — apply a blur effect
  - **HueRotate** — rotate the color hue
- 🌙 Dark mode UI
- ⚡ Instant live preview as you adjust sliders

## 🧑‍💻 How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/amanyhisham/Photo-Editor.git
   ```
2. Open `index.html` in your browser.
3. Click **Upload** to choose an image.
4. Use the sliders to edit your photo in real time!
5. Can Download on your device
